Steps to Run:
Navigate to Your Project Directory:

Open Command Prompt or Terminal.
Use the cd command to change directories to where your Flask app is located. For example:

cd C:\Users\hp\Desktop\reading material\New Projects\invoice system

To migrate new database
flask db init
flask db migrate -m "Initial migration."
flask db upgrade
